La navegación principal contiene botones con opciones para trabajar en el documento. Para pasar de un botón a otro, usa las teclas de flecha a la izquierda y flecha a la derecha; para activar un botón, usa la tecla Enter (Entrar). Para ir al editor, presiona “⌘F2”.
Lienzo
El lienzo es la página donde creas tu diseño. Para ir allí, presiona la tecla “Entrar”. Para ir a grupos o elementos en la página, vuelve a presionar la tecla “Entrar”. Presiona el tabulador para ir a la siguiente página, al siguiente grupo o al siguiente elemento. Para usar la barra de herramientas, usa la tecla “⌘F1”. Si ya seleccionaste un elemento, podrás moverlo usando las teclas de flechas, rotarlo con la teclas “,” y “.”, o cambiarle el tamaño con las teclas de flechas y la tecla “⌘”. También podrás aumentar los incrementos de estos atajos con la tecla “⇧”.
